Técnicas Usadas

Características Técnicas Exemplos
Layouts Usou-se o CSS grid e CSS flexbox para construção de layouts Utilizado em todas as páginas para posicionar as etiquetas HTML5
Responsavidade Foram usadas media queries que adaptam o layout para telemovel e PC consoante o tamanho da janela do browser Utilizado em todas as páginas, na página FAQ, com a utilização de media queries, quando o ecrã atinge uma certa dimensão a lista de Q&A passa de row para column; Tamanho da imagem resposivo
Seletores Foram usados os seguintes seletores: universal, tipo, classe, id, artibuto, pseudo-classe, pseudo-atributo Universal - aplicar a fonte a todo o código; Tipo - estilizar o header; Classe - estilizar o equilizer; Id - estilizar o menu; Atributo - estilizar o elemento 'a' que possui o atributo [target=_blank]; Pseudo-classe - estilizar o elemento 'a' quando o rato é clicado :active; Pseudo-elemento - ::first-line
Etiquetas HTML5 main, header, article, footer, figure, aside, nav; Utilizado em todas as páginas para definir a estrutura do site
Animações Foram feitas animações usando keyframes que permite alterar a opacidade da imagem no footer e alterar o tamanho de umas barras criando um equalizador Controlar a opacidade da imagem no sub_footer; Alterar o tamanho das barras para criar o efeito de movimento do equalizador
Efeitos Usou-se o efeito de opacidade, e o efeito hover overlay Opacidade utilizado na imagem do sub_footer; Hover Overlay utilizado na imagem de Download
Etiquetas Foi utilizado etiquetas para criar conteúdo como: figcaption, details, form, SVG, a, rect Wordcloud está dentro de uma etiqueta SVG com a sobreposicão de retângulos que possuem um link externo
