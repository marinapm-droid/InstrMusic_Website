About Us

InstrMusic is a college project created by two friends, that are music lovers, and one of them an amateur musician, that decided to work together to create an app that allows anyone to create music with their phone sensors. Select a sensor, select a sound, select an effect and start making your own music whenever you feel like it!

Supercollider

The magic happens in SuperCollider, a platform for audio synthesis used by musicians, artists and researchers working with sound. The software is totally free and open source! In the video, we can see two musicians creating live music with SuperCollider, and it's just one example of the many things that we can do with this programming language.
